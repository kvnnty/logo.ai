# Screenshots Directory

This directory should contain screenshots and images for the documentation.

## Required Files

- `logo.svg` - LogoAIpro logo (48x48px recommended)
- Screenshots of the application interface
- Setup process screenshots (optional)

## Logo

Place your LogoAIpro logo file as `logo.svg` in this directory. The logo should be:
- SVG format (recommended) or PNG
- 48x48px or larger
- Transparent background (if PNG)
